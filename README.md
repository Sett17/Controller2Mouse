# Controller2Mouse

Mapping:

| Controller     | PC               |
| -------------- | ---------------- |
| Left Stick     | MouseMove        |
| Left Trigger   | Mouse speedBoost |
| Left Stick in  | Middle Click     |
| Right Stick    | Scroll           |
| Right Stick in | Right Click      |
|                |                  |
| A              | Left Click, hold |
